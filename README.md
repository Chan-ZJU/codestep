## 提交标志含义

* 🚀 完结：整节课做完。
* 💥 大更新：大更新代表完成某个 lab 或者做完的一部分作业等。
* 🔥 小更新：小更新代表添加部分内容，修改某些细节等。

## 下载

```bash
git clone https://github.com/weijiew/codestep.git
cd codestep
yarn install
vuepress dev docs
```

查看：http://localhost:8080/ （端口可能不是 8080 ，注意查看终端输出！）
