## Q1: Product

```python
def product(n, f):
    sum = 1
    while n:
        sum *= f(n)
        n -= 1
    return sum
```

## Q2: Accumulate

注意运算次序，题目很简单！

```python
def accumulate(combiner, base, n, f):
    k = 1
    while k <= n:
        base = combiner(base, f(k))
        k += 1
    return base
```

写成这样虽然简单但是修改了运算次序，倒数第二个数据过不去。

```python
    while n > 0:
        base = combiner(base, f(n))
        n -= 1
    return base
```

## Q3: Make Repeater
