# 数组的顺序存储
数组：由相同类型的数据元素构成的有限集合。

一维数组可以看成线性表，二维数组起始就是数组套数组，只不过将一位数组中的数据想象成数组。

索引公式 = 第一个元素的地址 + 之后元素的个数 * 单个元素所占字节。

# 特殊矩阵的压缩存储
可以用数组来表示矩阵，但是表示的过程中有一些特殊的数据，为了节省空可以压缩存储。例如对称矩阵，三角矩阵，稀疏矩阵（非零元素小于 5% 的矩阵）等。直接存进去会浪费很多空间，根据数据的特性可以实现压缩。
